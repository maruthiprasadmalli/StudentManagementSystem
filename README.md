# Student Management System 🎓

A Java-based console application to manage student records using OOP concepts and MySQL database.

## 🛠️ Technologies Used
- Java 26
- MySQL 8.0
- JDBC Connectivity
- DAO Architecture Pattern

## 📌 OOP Concepts Used
- **Encapsulation** → Private fields with Getters/Setters
- **Inheritance** → Student extends Person
- **Polymorphism** → Method Overriding (displayInfo)
- **Abstraction** → Service Layer

## ✅ Features
- Add New Student
- View All Students
- Update Student Details
- Delete Student Record
